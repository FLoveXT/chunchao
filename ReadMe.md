#### 欢迎来到pangu微前端框架

* 乞丐版，仅仅能跑

#### 启用流程

* git clone https://github.com/JinJieTan/chunchao.git

* cd chunchao 

* yarn global add parcel-bundler

* yarn 

* yarn startSubApp1

* yarn startSubApp2 

* yarn start:main 

* 访问浏览器：http://localhost:1234/

> 恭喜你已经开启了chunchao的第一步了
